Sample Work

Sleep Quiz

This is a simple quiz app that tests users knowledge on sleep. Each question is presented as a multiple choice, requires the user to answer before progressing, and does not allow the user to retry questions. Users receive a success message or corrected answer, depending upon their selection, and their updated score after each submission. The final step presents the total score and the ability to restart the quiz.
